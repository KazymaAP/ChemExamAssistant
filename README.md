# ChemExam Assistant

Телеграм WebApp для подготовки к экзамену по химии.

## Что уже есть
- билеты, теория и практика
- тест и режим экзамена
- заметки, закладки, статистика, достижения
- персонализация (темы, шрифт, акцент)

## Быстрый запуск
```bash
python3 -m http.server 4173
# открыть http://localhost:4173
```

## Что улучшено в последней итерации
- добавлен экран **«Слабые темы»** с приоритизацией сложных вопросов
- добавлена кнопка **«Продолжить»** с переходом к последнему открытому вопросу
- добавлен простой кэш загрузки данных (`loadAllData(force = false)`), чтобы избежать лишних чтений localStorage
- в режиме экзамена ведется счетчик ошибок по вопросам (`stats.questionMistakes`)
